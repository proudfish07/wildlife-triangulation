# wildlife-triangulation

本專案提供一套野生動物定位與三角測量工具，支援離線使用，特別設計給保育人員與研究者在野外缺乏網路的環境中進行精準紀錄與追蹤。

# 📦 專案特色
- 離線運作：無需網路即可使用（底圖除外）
- 多平台支援：Android 與 iOS 手機皆可使用
- 全螢幕地圖介面：操作按鈕懸浮於地圖上，觀測資料收納於底部上拉面板
- GPS 自動定位與即時定位追蹤
- 支援方位角紀錄（含指南針讀取與磁偏角修正）與多點三角測量
- 觀測資料、個體編號、備註與計算結果自動保存於裝置，重開仍在
- 適合野外調查、保育紀錄與教育用途

# 📊 三角測量法
本應用採用幾何三角定位法（Geometric Triangulation）來推估動物發報器的所在位置。這種方法運算快速、邏輯直觀，特別適合野外調查人員於現場進行初步位置判斷與搜尋範圍規劃。

## 🔧 基本原理
調查人員在多個位置點記錄下：
- GPS 定位座標（經緯度）
- 接收到訊號的方位角（以度為單位，0° 為正北，順時針方向）
- （可選）增益（Gain）與訊號強度（Signal Strength）

每一組紀錄可視為一條射線，方向對應於方位角。藉由多條射線的交會點，可推估訊號來源的可能位置。這些交會點之間的分布，反映了量測誤差與方向準確性。

## 🧪 計算公式

lat_avg = (1/n) × (lat₁ + lat₂ + ... + latₙ)

lon_avg = (1/n) × (lon₁ + lon₂ + ... + lonₙ)

latᵢ, lonᵢ：第 i 個交點的緯度與經度

lat_avg, lon_avg：所有交點的平均值（即理論預測位置）

# 📌 進階定位建議
本應用所提供的計算結果，已能作為現場初步搜尋的依據。但若調查任務需更精確的估位結果（例如用於科學研究或長期監測），建議可將觀測資料匯出並使用以下專業軟體進行後續分析：
- LOAS – 無線電追蹤與估位常用桌面工具
- sigloc R 套件 – 基於方位角與強度進行空間估算的 R 工具包
- Locate III – 針對動物無線電追蹤設計的進階定位軟體

# 📱 如何在手機端安裝與使用（Android / iOS）

## Android 使用者 
1. 使用 Chrome 或 Firefox 開啟專案頁面：https://proudfish07.github.io/wildlife-triangulation
2. 點選右上角選單（⋮）> 選擇「安裝應用程式」或「加入主畫面」
3. App 會以全螢幕方式啟動，並可離線使用

## iOS 使用者
1. 使用 Safari 瀏覽器開啟：https://proudfish07.github.io/wildlife-triangulation
2. 點選下方「分享」圖示 > 選擇「加入主畫面」
3. 點擊加入後，主畫面會出現 App 圖示，可像原生 App 一樣使用

首次載入後資料會快取，可於離線狀態下使用

# 📖 使用說明（手機端操作流程）
本應用為全螢幕地圖介面：主要操作為地圖上的四顆懸浮圓形按鈕；輸入欄位與觀測資料表收納在螢幕底部的上拉面板中（點擊把手或向上滑動即可展開）。

## 🔘 懸浮按鈕說明（地圖下方，由左至右）
| 按鈕 | 功能說明 |
| ---- | ------- |
| ⌖ **自動定位** | 透過手機 GPS 自動取得目前經緯度座標，並加入一筆新的偵測紀錄（日期、時間自動填入）。需允許定位權限。定位中圖示會旋轉。 |
| ((·)) **即時定位** | 開關式按鈕。開啟後持續更新地圖上代表你所在位置的藍點（按鈕變綠色並有光圈提示）；再按一次關閉以節省電力。 |
| ✕ **交點計算** | 對目前所有紀錄的方位角資料進行三角測量，推估發報器個體可能所在的位置，並將地圖縮放至交會點範圍。 |
| **分享報表** | 以系統分享功能送出觀測紀錄（含個體編號、備註、各筆觀測資料、預測位置與 Google 地圖連結）；無系統分享功能的環境（如桌面瀏覽器）會自動改為複製到剪貼簿。 |

## 📋 底部上拉面板內容
- **目前追蹤個體編號**：輸入後自動記憶於裝置，重開網頁不需重填；清除資料時亦會保留
- **偏角修正（磁北轉真北）**：預設 -4.5°，用於指南針讀值修正
- **備註**：整次追蹤的補充說明（天氣、地形、訊號干擾等），會隨報表一起分享；清除資料時一併清空
- **清除資料**：清除所有觀測紀錄、備註與計算結果（個體編號保留）。請小心使用
- **觀測資料表**：可左右滑動，逐筆修改方位角（🧭 鈕可直接讀取手機指南針）、Gain、強度、活動狀態，或刪除單筆紀錄

## 🧭 操作流程步驟
1. 收到發報器訊號後，進入本應用並允許定位權限。
2. 點選「自動定位」按鈕，自動加入一筆含目前時間與經緯度的紀錄。
3. 展開底部面板，於該筆紀錄填入：
   - 方位角（必填）：以度為單位，0° 表示正北，順時針方向；可按 🧭 由手機指南針自動讀取（已含磁偏角修正）。
   - Gain 值（選填）與訊號強度（選填）。
4. 前往下一個偵測位置後，重複步驟 2–3。
5. 建議至少蒐集 3–5 筆觀測資料。
6. 點選「交點計算」，地圖會標示預測位置並自動縮放至交會點範圍。
7. 點選「分享報表」，將資訊傳送給其他人作紀錄與行動參考。計算結果會保存於裝置，重開網頁後分享仍會帶出。

## ⚠️ 注意事項
- 📡 底圖需網路連線才能正常載入，但即使離線狀態下仍可使用 GPS、自動定位與交點計算等功能。
- 🛰️ 使用前請確保裝置已開啟定位功能，且授權瀏覽器使用 GPS。
- 🔒 報表內容含精確座標，轉發前請留意資訊安全與物種保育考量。
